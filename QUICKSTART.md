## 🎬 How to Use

### View Locally
1. Open the portfolio folder
2. Double-click `index.html` to open in your browser
3. Or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit: http://localhost:8000
   ```

### Print to PDF
1. Click the 🖨️ Print button in the hero section
2. Or press Ctrl+P (Windows) / Cmd+P (Mac)
3. Select "Save as PDF" and choose your location

### Deploy Online
You can easily deploy this portfolio to:
- **GitHub Pages** - Free hosting
- **Netlify** - Drag & drop deployment
- **Vercel** - Serverless platform
- **Any web host** - Simple FTP upload

## 📝 Customization Tips

### Change Colors
Edit `css/main.css` and modify:
```css
:root {
    --primary-color: #0d47a1;      /* Main blue */
    --secondary-color: #00838f;    /* Teal accent */
    --accent-color: #ffeb3b;       /* Yellow highlights */
}
```
