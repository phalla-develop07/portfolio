## 🔧 Customization Options

### Easy Changes
1. **Update CV data** - Edit `js/data/` files

4. **Modify animations** - Adjust timing in `css/animations.css`

### Advanced Changes
1. **Add new sections** - Create new data file + section CSS

4. **New components** - Create in `css/components/`
